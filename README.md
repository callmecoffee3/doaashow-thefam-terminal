### **Features**

| Section | Commands | Notes |
| --- | --- | --- |
| **Posts** | `post <text> [public\|private]` | Main feed, searchable |
| **Video/Photo** | `video post <url> [public\|private]` | Media feeds |
| **Messenger** | `dm <user> <msg>` | Always private |
| **Email** | `email send <user> <subj> <body>` | Private inbox |
| **Pages** | `page create <name>` | Brand/public pages |
| **Family** | `family create <name>` | Private group - members see your private posts |
| **Business** | `business create <name> <desc>` | Business profiles |
| **Boards** | `board create <name>` | Topic boards like Reddit |
| **Search** | `search <section> <query>` | Searches all content you can see |
| **Incognito** | `incognito on/off` | Post as "Anonymous" |

**Privacy Model**: `public` = everyone sees it. `private` = only you + your family members. `incognito on` = public posts show as Anonymous.
